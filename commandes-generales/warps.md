# 🌀 Warps

Les warps sont des points de téléportation globaux accessibles par tous les joueurs. 🗺️

---

## 📋 Commandes

| Commande       | Description                                    |
| -------------- | ---------------------------------------------- |
| `/warps`       | 📋 Affiche la liste de tous les warps disponibles |
| `/warp <nom>`  | 🌀 Te téléporte au warp indiqué                  |

{% hint style="info" %}
🌐 Les warps sont partagés entre **tous les serveurs**. Tu peux te téléporter vers un warp même s'il est sur un autre shard !
{% endhint %}

---

## 📋 Warps disponibles

Les warps sont gérés par l'équipe du serveur. Tu peux retrouver la liste complète en tapant `/warps` en jeu. 🎮
