# 👥 Membres & Coopération

Joue en équipe ! 🤝 Invite tes amis sur ton île ou accorde des accès temporaires.

---

## 🤝 Inviter des membres

| Commande                      | Description                              |
| ----------------------------- | ---------------------------------------- |
| `/is invite <joueur>`         | 📨 Invite un joueur à rejoindre ton île   |
| `/is accept <propriétaire>`   | ✅ Accepte une invitation d'île            |
| `/is kick <joueur>`           | 🚫 Expulse un membre de ton île           |
| `/is leave`                   | 🚪 Quitte l'île dont tu es membre         |
| `/is team`                    | 👥 Ouvre le menu de gestion d'équipe      |

{% hint style="success" %}
💡 Le menu `/is team` te permet de voir et gérer les **rôles de chaque membre** visuellement !
{% endhint %}

---

## 🔓 Système Coop (accès temporaire)

Le coop permet de donner un accès temporaire à un joueur **sans l'ajouter comme membre permanent**. ⏳

| Commande              | Description                          |
| --------------------- | ------------------------------------ |
| `/is coop <joueur>`   | 🔓 Accorde l'accès coop à un joueur  |
| `/is uncoop <joueur>` | 🔒 Retire l'accès coop               |
| `/is cooplist`        | 📋 Affiche la liste des joueurs en coop |

{% hint style="info" %}
ℹ️ **Aliases** : `/is trust` et `/is untrust` fonctionnent aussi !
{% endhint %}

---

## 🛡️ Permissions

| Commande            | Description                                |
| ------------------- | ------------------------------------------ |
| `/is permissions`   | 🛡️ Ouvre le menu de gestion des permissions |

Depuis ce menu, tu peux configurer ce que chaque rôle (**membre**, **coop**, **visiteur**) peut faire sur ton île. ⚙️
