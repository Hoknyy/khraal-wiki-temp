# Banque & Économie

Chaque île possède sa propre banque partagée entre les membres.

---

## Commandes banque

| Commande | Description |
| --- | --- |
| `/is bank` | Affiche le solde de la banque d'île. |
| `/is bank deposit <montant>` | Dépose de l'argent dans la banque. |
| `/is bank withdraw <montant>` | Retire de l'argent de la banque. |
| `/is bank history` | Consulte l'historique des transactions. |

{% hint style="info" %}
L'historique est paginé : utilise `/is bank history 2` pour voir la page suivante.
{% endhint %}

---

## Valeur de l'île

| Commande | Description |
| --- | --- |
| `/is value` | Affiche la valeur totale de ton île. |
| `/is value refresh` | Recalcule la valeur de ton île. |

{% hint style="success" %}
La valeur de ton île dépend des blocs et spawners que tu as placés. Plus ta valeur est élevée, plus tu montes dans le classement.
{% endhint %}
