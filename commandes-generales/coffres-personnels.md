# 📦 Coffres Personnels (Vaults)

Tes coffres personnels te suivent partout ! 🌍 Ils sont synchronisés entre **tous les serveurs**.

---

## 📋 Commandes

| Commande              | Description                               |
| --------------------- | ----------------------------------------- |
| `/pv`                 | 📦 Ouvre ton coffre personnel #1           |
| `/pv <numéro>`        | 🔢 Ouvre le coffre numéro N                |
| `/pv list`            | 📋 Affiche la liste de tes coffres non vides |
| `/pv delete <numéro>` | 🗑️ Supprime le contenu d'un coffre         |

{% hint style="info" %}
💡 **Aliases disponibles** : `/chest`, `/vault`, `/playervaults`
{% endhint %}

---

## 🎒 Taille et nombre de coffres

Le nombre de coffres et leur taille dépendent de ton grade :

| Grade   | Nombre de coffres | Taille             |
| ------- | ----------------- | ------------------ |
| Joueur  | 1                 | 1 ligne (9 slots)  |
| VIP     | 3                 | 3 lignes (27 slots)|
| VIP+    | 5                 | 4 lignes (36 slots)|
| MVP     | 10                | 6 lignes (54 slots)|

{% hint style="warning" %}
⚠️ Tu ne peux pas ouvrir le même coffre sur **deux serveurs en même temps** (protection anti-duplication).
{% endhint %}
