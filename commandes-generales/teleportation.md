# 🧭 Commandes de Téléportation

Déplace-toi rapidement à travers les shards du serveur ! ✨ Toutes ces commandes fonctionnent **entre les serveurs**.

---

## 📍 Spawn & Retour

| Commande  | Description                                              |
| --------- | -------------------------------------------------------- |
| `/spawn`  | 🌟 Te téléporte au spawn du serveur                      |
| `/back`   | ↩️ Retourne à ta position précédente (mort ou dernier TP) |

---

## 🏠 Homes

Sauvegarde tes lieux préférés et téléporte-toi en un clic ! 🖱️

| Commande          | Description                           |
| ----------------- | ------------------------------------- |
| `/sethome [nom]`  | 📌 Crée un home à ta position actuelle |
| `/home [nom]`     | 🏠 Te téléporte à un de tes homes      |
| `/delhome <nom>`  | 🗑️ Supprime un home                    |

{% hint style="info" %}
💡 Le nombre de homes dépend de ton grade. Par défaut tu en as **1** !
{% endhint %}

### 📊 Limites de homes par grade

| Grade   | Nombre de homes |
| ------- | --------------- |
| Joueur  | 1               |
| VIP     | 3               |
| VIP+    | 5               |
| MVP     | 10              |

---

## 👥 Téléportation entre joueurs

| Commande          | Description                                        |
| ----------------- | -------------------------------------------------- |
| `/tpa <joueur>`   | 📨 Demande à te téléporter vers un joueur           |
| `/tphere <joueur>`| 📩 Demande à un joueur de venir vers toi            |
| `/tpaccept`       | ✅ Accepte une demande de téléportation              |

{% hint style="success" %}
🌐 Ces commandes fonctionnent même si le joueur est sur un **autre shard** !
{% endhint %}
