# Améliorations & Quêtes

Fais évoluer ton île et accomplis des quêtes pour gagner des récompenses.

---

## Améliorations (Upgrades)

| Commande | Description |
| --- | --- |
| `/is upgrades` | Ouvre le menu des améliorations. |
| `/is border` | Configure la bordure de ton île. |
| `/is limit` | Affiche les limites de blocs de ton île. |

{% hint style="info" %}
Les upgrades te permettent d'agrandir ton île, augmenter tes limites de blocs, améliorer ta bordure et bien plus.
{% endhint %}

---

## Quêtes

| Commande | Description |
| --- | --- |
| `/is quest` | Ouvre le menu des quêtes. |

Les quêtes sont des objectifs à accomplir sur ton île. Elles te rapportent de l'expérience, de l'argent et des objets rares.

---

## Battle Pass

| Commande | Description |
| --- | --- |
| `/is pass` | Ouvre le Battle Pass. |

Le Battle Pass est un système de progression saisonnier. Accomplis des défis pour débloquer des paliers et récupérer des récompenses exclusives.

{% hint style="warning" %}
Le Battle Pass se réinitialise chaque saison. N'oublie pas de récupérer tes récompenses.
{% endhint %}
