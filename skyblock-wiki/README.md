# Bienvenue sur le Wiki de Khraal Skyblock !

Tu viens d'arriver sur le serveur ? Ce wiki est fait pour toi. Retrouve ici toutes les commandes et astuces pour bien débuter ton aventure.

---

## Ce que tu trouveras ici

{% hint style="info" %}
Ce wiki couvre les commandes essentielles pour jouer sur Khraal. Il est mis à jour régulièrement.
{% endhint %}

### Commandes générales

Toutes les commandes utiles au quotidien : téléportation, homes, warps, coffres personnels.

### Skyblock

Tout pour gérer ton île : création, membres, banque, upgrades, quêtes.

---

## Pour commencer

| Étape | Commande | Description |
| --- | --- | --- |
| 1 | `/is create` | Crée ton île Skyblock. |
| 2 | `/sethome base` | Pose un home sur ton île. |
| 3 | `/is upgrades` | Découvre les améliorations. |
| 4 | `/pv` | Ouvre ton coffre personnel. |

{% hint style="success" %}
Besoin d'aide en jeu ? Utilise `/help` ou demande dans le chat.
{% endhint %}
