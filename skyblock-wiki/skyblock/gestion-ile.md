# Gestion de ton île

Tout commence par une île. Voici les commandes essentielles pour créer et gérer ton île Skyblock.

---

## Commencer l'aventure

| Commande | Description |
| --- | --- |
| `/is create` | Crée ta nouvelle île Skyblock. |
| `/is home` | Te téléporte sur ton île. |
| `/is help` | Affiche l'aide des commandes d'île. |

{% hint style="info" %}
Aliases : tu peux aussi utiliser `/island`, `/sb` ou `/skyblock` au lieu de `/is`.
{% endhint %}

---

## Gérer ton île

| Commande | Description |
| --- | --- |
| `/is name <nom>` | Renomme ton île (max 32 caractères). |
| `/is delete` | Supprime ton île (confirmation requise). |
| `/is sethome` | Change le point de spawn de ton île. |
| `/is chat` | Active ou désactive le chat d'île pour les membres. |

---

## Warps d'île

Crée des points de téléportation sur ton île pour tes visiteurs.

| Commande | Description |
| --- | --- |
| `/is setwarp <nom>` | Crée un warp sur ton île. |
| `/is warp [nom]` | Te téléporte à un warp d'île. |
| `/is delwarp <nom>` | Supprime un warp d'île. |

{% hint style="info" %}
Sans nom, `/is warp` affiche la liste des warps disponibles.
{% endhint %}
