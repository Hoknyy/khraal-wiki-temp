# Wiki staff KhraalGames

Ce wiki regroupe les procédures réservées aux modérateurs et administrateurs du réseau KhraalGames.

{% hint style="warning" %}
Ce contenu est interne au staff. Ne pas recopier d'informations techniques, procédures d'enquête ou commandes sensibles dans le wiki joueur.
{% endhint %}

## Périmètre modération

Les modérateurs sont là pour inspecter les comportements suspects, modérer le chat, aider les joueurs et appliquer les sanctions prévues. Ils ne doivent pas utiliser d'outils qui donnent des items, valident des quêtes, modifient l'économie, changent fortement le gameplay ou contournent une progression joueur.
